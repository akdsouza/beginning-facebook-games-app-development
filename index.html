Welcome to Beginning Facebook Apps Development.

Welcome to the wild world of browser-based games development for Facebook! This is the companion code for the book Beginning Facebook Game Apps Development. To get started, simply clone the repo: 

$ git clone git://github.com/waynegraham/beginning-facebook-games-app-development.git
$ cd beginning-facebook-games-app-development

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
